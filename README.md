# 📖 [Calculator] <a name="about-project"></a>

A **Simple Calculator**  is a web app that is designed to perform basic arithmetic operations, such as addition, subtraction, multiplication, and division. The app typically displays a display that shows the current value of the calculator, along with a row of buttons that represent the different arithmetic operations.
